# Rekening Semantik Umum

## 1. Pendahuluan

Rekening Semantik Umum merupakan struktur path akun yang bersifat **universal**, dapat digunakan oleh seluruh jenis usaha, dan berfungsi sebagai fondasi utama sistem Rekening Semantik Less Summer.

Setiap akun ditulis secara **deskriptif dan hierarkis**, dengan struktur:

```markdown

[rekening].[kategori].[subkategori].[detail].[spesifik]

```

Struktur ini memudahkan integrasi lintas sistem — laporan keuangan, pajak, maupun analitik.

## 9. Total Jumlah Path (≈75+)

Terdapat lebih dari **75 path akun umum** yang mencakup semua elemen dasar akuntansi.  
Struktur ini dapat digunakan sebagai *core account list* di:
- ERP dan software akuntansi internal,  
- laporan keuangan PSAK,  
- laporan pajak (PPh & PPN), serta  
- sistem analitik bisnis dan pelaporan BI.

---

## 10. Prinsip Umum Rekening Semantik

1. **Konsistensi Penamaan**  
   Gunakan pola `[rekening].[kategori].[subkategori].[detail].[spesifik]` secara ketat.  

2. **Deskriptif dan Hierarkis**  
   Setiap segmen menjelaskan fungsi akun dan dapat diurai hingga level granular.

3. **Tag Fiskal**  
   Gunakan `.final`, `.non_final`, `.deductible`, `.non_deductible` untuk mempermudah pelaporan pajak.

4. **Modular dan Skalabel**  
   Dapat diperluas untuk domain industri tertentu tanpa mengubah struktur inti.

5. **Keterhubungan Lintas Sistem**  
   Setiap path dapat langsung dipetakan ke laporan keuangan, pajak, maupun API akuntansi.

## 11. Contoh Transaksi

| Transaksi | Debit | Kredit |
| :-- | :-- | :-- |
| Penjualan tunai | `assets.cash.bank.bca` | `income.sales.product.retail.non_final` |
| Pembayaran gaji | `expenses.operational.payroll.salary.deductible` | `assets.cash.bank.bca` |
| Pembayaran listrik | `expenses.operational.utilities.electricity.deductible` | `assets.cash.bank.mandiri` |
| Pembayaran PPh 25 | `assets.prepaid.tax.pph25` | `assets.cash.bank.bca` |
| Pengakuan PPh 25 sebagai kredit pajak | `liabilities.tax.payable.pph25` | `assets.prepaid.tax.pph25` |
| Penyusutan laptop | `expenses.depreciation.equipment.laptop.deductible` | `assets.fixed.accumulated_depreciation` |

## 12. Kesimpulan

Rekening Semantik Umum adalah **inti dari sistem akuntansi modern** Less Summer — menggabungkan kejelasan semantik, konsistensi, dan kepatuhan fiskal.  
Struktur ini dapat menjadi *lingua franca* bagi laporan keuangan, pajak, dan analitik lintas sistem tanpa memerlukan interpretasi tambahan.

> Semua rekening khusus industri (misalnya konstruksi, properti, atau digital) akan dikembangkan di atas standar umum ini,  
> memastikan kompatibilitas penuh dan keterbacaan universal.
